<p>react-navigation v4 で使える hooks api に新しく hooks が追加されました。</p>
<ul>
<li>useFocusEffect</li>
<li>useIsFocused</li>
</ul>
<h2>useFocusEffect</h2>
<p><code>useFocusEffect</code> は使っている Screen に Focus があたった時、及び Focus が外れた時にサイドエフェクトを起こします。
<code>useEffect</code> の Screen 版のようなものですね。</p>
<pre><code class="language-jsx">function MyScreen() {
  useFocusEffect(
    useCallback(() => {
      console.debug('screen takes focus')
      return () => console.debug('screen loses focus')
    }, [])
  )
  return &#x3C;View>...&#x3C;/View>
}
</code></pre>
<h2>useIsFocused</h2>
<p><code>useIsFocused</code> は現在 screen が focus されているか否かを boolean で返します。</p>
<pre><code class="language-jsx">function MyScreen() {
  const isFocused = useIsFocused()
  return &#x3C;Text>{isFocused ? 'Focused' : 'Not Focused'}&#x3C;/Text>
}
</code></pre>
<p>react navigation の stack は mount が発火されるタイミングがわかりにくいので、screen 毎の<code>useEffect</code>のような hooks がが使える様になったのはありがたいですね。</p>
<p>なお、次に来る react-navigation v5 はフルリライトでガラッと使い方が変わるので、この hooks を活用出来る期間はそう長くはないかもしれません。</p>
